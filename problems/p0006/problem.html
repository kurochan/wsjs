単語リストの生成2

単語リストを作りましょう。
文中に出てきた単語を、頻度の高い順に出力してください。その際、空白を一つあけて、その単語の出現回数を出力してください。
同じ回数の単語があった場合は、辞書順で先にくる方の単語を先に出力してください。

サンプル
入力 
java lisp ruby lisp nasm
atom bash atom curl ruby atom
nasm ruby java sh tex
出力 
atom 3
ruby 3
java 2
lisp 2
nasm 2
bash 1
curl 1
sh 1
tex 1
ヒント キーと値をセットで記憶できるデータ構造があったはずです。
キーに単語、値に単語の出現回数を格納してみましょう。
